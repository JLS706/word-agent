## 推荐学习路线

```
=== Agent 项目 ===
Day 1: 01-架构 + 02-ReAct → 理解全貌
Day 2: 03-Multi-Agent + 07-错误 → 掌握流水线
Day 3: 06-记忆技能 + 04-沙盒 + 05-工具 → 精通子系统
Day 4: 11-语义分块 + 08-八股文 → 补齐通用知识

=== 秒杀项目 ===
Day 5: 12-项目概述 → 练口述模板（3 分钟能说清链路）
Day 6: 13-秒杀核心 → 超卖、Redis、锁、MQ（被追问最多的部分）
Day 7: 14-八股补强 → 类加载、ThreadLocal、索引、并发

=== 面试冲刺 ===
Day 8: 09-简历 + 10-面试QA → 针对性模拟
Day 9: 15-白板手册 → 对着白板手册练画图（能脱稿画出三大引擎骨架）
```
